# 📊 Unemployment Analysis using Python

This project is part of my Data Science Internship at CodeAlpha.

## 📌 Project Overview
The objective of this project is to analyze unemployment trends using Python and derive meaningful insights from the data.

The analysis focuses on understanding patterns, regional differences, and the impact of Covid-19 on unemployment rates.

---

## 📂 Dataset
The dataset contains information about unemployment rates across different regions and time periods.

Key columns include:
- Date
- Region
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib

---

## 🔍 Key Steps Performed
- Data Cleaning (handling missing values, formatting columns)
- Exploratory Data Analysis (EDA)
- Visualization of unemployment trends
- Region-wise unemployment comparison
- Covid-19 impact analysis
- Monthly trend analysis

---

## 📈 Key Insights
- Unemployment rates vary significantly across regions
- There is a noticeable increase during the Covid-19 period
- Monthly trends show fluctuations in unemployment patterns

---

## 🚀 Conclusion
This project demonstrates how data analysis techniques can be used to understand real-world economic problems and extract valuable insights.
